SlidyPages

Full page presentation-like with a vertical scroll

SlidyPages allows you to create full pages with a vertical scroll by using the script and CSS with a basic HTML model.

The basic HTML, the script and the CSS stylesheet can be found on the github repo.

You can use different backgrounds per page, like all the pages so far...

Also, you can set a background color for the body, and it will appear in every page without a set color, like this and...

... this! I have the same color as the body and the previous page!

The slidy-container class has an attribute called "attr-page", which tracks the current page.

You can use this to customize the navigation and pagination buttons in certain pages with CSS. Check it on the next page!

This page changes the "go up" button to gold and adds an animation to it.

The pagination selector is set to red on the unselected pages and orange to the selected one.

By the way, this page and the last one also share a common continuous background.
